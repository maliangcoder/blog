# 递增和递减运算符

  • 如果需要反复给数字变量添加或减去1,可以使用递增(++)和递减(--)运算符来完成
  • 在JavaScript中,递增(++)和递减(--)既可以放在变量前面,也可以放在变量后面,放在变量前面时,可以称之为前置递增(递减)运算符,放在变量后面时,可以称之为后置递增(递减)运算符
  • 注意:递增和递减运算符必须和变量配合使用
  1. 前置递增运算符
    • ++num前置递增,就是自家1,类似于num = num + 1,但是 ++num写起来更简单
    • 先自加,再参与运算(后返回值)
     var age = 10;
     ++age; // 类似于 age = age + 1

     /* 先加1 后返回值 */
     var p = 10;
     console.log(++p + 10) //21

# 比较运算符

  • 比较运算符，是两个数据进行比较时所使用的运算符，比较运算后，会返回一个布尔值(true/false)作为比较运算的结果。

# 逻辑运算符

  • 逻辑运算符是用来进行布尔值运算的运算符，其返回值也是布尔值。
    • && 逻辑与
      • 如果两侧都为true，结果才是true，是要有一侧为false，结果就为false
    • || 逻辑或
      • 两侧都为false，结果为false，只要有一测为true，结果就是true
    • ! 逻辑非
      • 取反
  • 短路运算(逻辑中断)

    1. 逻辑与

      • 语法:表达式1 && 表达式2
      • 如果第一个表达式的值为真，则返回表达式2
      • 如果第一个表达式的值为假，则返回表达式1

    2. 逻辑或

      • 语法:表达式1 || 表达式2
      • 如果表达式1结果为真 则返回的是表达式1
      • 如果表达式1结果为假 则返回的是表达式2
