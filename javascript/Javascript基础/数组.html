# 遍历数组

  • 就是把数组中的元素从头到尾都访问一次
  • 可以通过数组名.length获取到数组的长度
var arr = ['a','b','c'];
for(var i = 0; i < arr.length; i++){
     console.log(arr[i])
}

  • 反转数组
    • 要求：不使用数组中的方法，纯代码实现
/* 反转数组 */
var arr = [1,2,3,4,5];
var newArr = [];
for(var i = arr.length - 1; i >= 0; i--){
     /* 实现思路，利用倒叙方式，此时的`i`是arr数组最后一位下标，
     把arr[i] 赋值给newArr，newArr下标就用自己的length来做下标，
     次循环过后newArr下标都会增加一位。 */
     newArr[newArr.length] = arr[i]
}

  • 数组排序(冒泡排序)
    • 把一系列的数据按照一定的顺序进行排列显示(从小到大或者从大到小)
/* 
     给定一个数组，实现反转
 */
 var arr = [5,4,3,2,1];
//  利用冒泡排序法
for(var i = 0; i<arr.length - 1; i++){
     for(var j = 0; i < arr.length - i - 1; j++){
          if(arr[j] > arr[j + 1]){
               var temp = arr[j];
               arr[j] = arr[j + 1];
               arr[j + 1] = temp;
          }
     }
}
