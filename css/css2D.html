# 2D转换

  • transform是CSS3中具有颠覆性的特征之一,可以实现元素的位移、旋转、缩放等效果

    • 2D转换之移动translate

      • 语法: transform:translate(x,y); x轴和y轴分开写 transform:translateX(x); 只移动x轴,x大写 transform:translateY(y) 只移动y轴,y大写

      • 注意:

        1. 定义2D转换中那个的移动,沿着x和y轴移动元素
        2. translate最大的优点:不会影响到其他元素的位置
        3. translate中的百分比单位是相对于自身元素的宽度和高度一半来对比的
        4. 对行内标签没有效果

      • 利用transform让盒子实现垂直水平居中

        .box{
     position:absolute;
     width:200px;
     height:200px;
     <!-- 利用了translate(x,y)百分比的单位是自身元素的一半实现垂直水平居中 -->
     transform:translate(-50%,-50%)
}

    • 2D转换之旋转rotate

    • 语法: - transform:rotate(度数)

      • rotate里面跟度数,单位是deg,比如rotate(45deg)
      • 角度为正时,顺时针,为负时,逆时针
      • 默认旋转的中心点时元素的中心点

    • 2D转换中心点transform-origin

      • 语法:transform-origin:x y
      • 重点:
        • 注意后面的参数x和y用空格隔开
        • x y默认转换的中心点时元素的中心点(50% 50%)
        • 还可以给x y设置像素或者方位名词(top bottom left right)

    • 2D转换之缩放scale

      • 语法:transform:scale(x,y);
      • 注意:
        • 注意其中x,y用逗号分隔
        • transform:scale(1,1); 宽和高都放大一倍,相对于没有放大
        • transform:scale(2,2); 宽和高都放大了2倍
        • transform:scale(2); 只写一个参数,第二个参数则和第一个参数一样,相当于scale(2,2)
        • transform:scale(0.5,0.5) 缩小
        • scale缩放最大的优势:可以设置转换中心点缩放,默认以中心点缩放的,而且不影响其他盒子

    • 注意:

      1. 同时使用多个转换,格式为:transform:translate() rotate() scale() ...
      2. 其顺序会影响转换的效果(先旋转会改变坐标轴方向)
      3. 同时有位移和其他属性的时候,要将位移放到最前
