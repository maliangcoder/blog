# 伪元素和结构伪类选择器

# 伪元素选择器

  • 语法: element::before{}element::after{}
    • before和after创建一个元素,但是属于行内元素
    • before和after必须有content属性
    • before在父元素内容的前面创建元素,after在父元素内容的后面插入元素
    • 伪元素选择器和标签选择器一样,权重为1

# 结构伪类选择器

     E:first-child
     E:last-child
     E:nth-child
     E:nth-of-type
1
2

  • 结构伪类选择器一般用于选择父级里面的弟几个孩子
  • nth-child 对父元素里面所有孩子排序选择(序号是固定的)先找到弟n个孩子,然后看看是否和E匹配
  • nth-child 对父元素立案指定子元素进行排序选择,先去匹配E,然后再根据E去找弟n个孩子
  • 关于nth-child(n) 我们要知道n是从0开始计算的,记住常用的公式
  • 如果是无序列表,我们一般用nth-child更多
  • 类选择器、属性选择器、伪类选择器权重都是10。
