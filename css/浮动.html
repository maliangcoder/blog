# 浮动（float）

  • 标准流(普通流/文档流)

    • 所谓标准流就是标签按照规定好默认方式排列。
    1. 块级元素会独占一行,从上向下顺序排列。 常用元素: div、hr、p、h1~h6、ul、ol、dl、form、table
    2. 行内元素会按照顺序,从左到右顺序排列,碰到父元素边缘则自动换行。 常用元素: span、a、i、em

    • 为什么需要浮动

      • 有很多的排列布局,标准流没有办法完成,此时就可以利用浮动完成布局。因为浮动可以改变元素标签默认的排列方式。

    • 网页布局第一准则:多个块级元素纵向排列找标准流,多个块级元素横向排列找浮动。

    • 语法: 选择器 { float:left|right }

    • 设置了浮动的元素最重要特性:

      1. 脱离标准普通流的控制,移动到指定位置
      2. 浮动的盒子不再保留原先的位置

    • 清除浮动

      1. 额外标签法

        • 再浮动元素末尾添加一个空的标签,比如<div style='clear:both'></div>,或者其他标签(如
          等)。 优点:通俗易懂,结构优雅 缺点:增加无意义的额外标签。

      2. 父级添加overflow 给父级添加overflow属性,将其属性值设置为hidden、auto、scroll。 优点:代码简洁。 缺点:无法显示溢出的部分。

      3. :after伪元素法

        .clearFix:after{
     content:'';
     display:block;
     height:0;
     clear:both;
     visibility:hidden;
}
.clearFix{
     *zoom:1;
}
